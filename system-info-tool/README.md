# System Information Tool

A Python script that gathers system details including OS, CPU, RAM, disk usage, hostname, and IP address.

## Features
- Detects operating system and version
- Shows CPU model and core count
- Displays total and available RAM
- Reports disk usage
- Retrieves hostname and IP address

## How to use
1. Install dependencies:
2. Run 'main.py'
3. View system information in the terminal
