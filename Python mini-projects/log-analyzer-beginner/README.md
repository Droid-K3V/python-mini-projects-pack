# Beginner Log Analyzer

A simple Python script that reads a log file, counts log levels, extracts timestamps, and prints a clean summary.

## Features
- Reads a '.log' file line by line
- Counts INFO, WARNING, and ERROR entries
- Extracts first and last timestamps
- Outputs a summary of log activity

## How to Use
1. Place your log file in the project folder
2. Make sure it's named 'sample.log'
3. Run 'main.py'
4. View the summary in the terminal

## Example Output
- Total log entries
- Count of each log level
- First and last timestamps
