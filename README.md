# Python Mini Projects Pack

A collection of three beginner-friendly Python projects designed to build core programming fundamentals and confidence. Each project includes clean code, documentation, and screenshots showing the program running.

---

## 📁 Included Projects

### 1. **Password Generator**
A simple tool that creates secure random passwords using letters, numbers, and symbols.

**Skills Learned:**
- Randomization
- String manipulation
- User input

---

### 2. **Calculator**
A basic calculator that performs addition, subtraction, multiplication, and division.

**Skills Learned:**
- Functions
- Conditionals
- Input handling
- Error checking

---

### 3. **File Organizer**
An automation script that sorts files into folders based on their file type.

**Skills Learned:**
- File handling
- Working with directories
- Automation logic
- Dictionaries and loops
